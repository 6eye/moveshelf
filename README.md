# moveshelf

Moveshelf is a cloud-based 3D digital motion visualization service and collaboration tool, 
created to make human motion data more accessible and easier to use. Find out more about [Moveshelf](https://moveshelf.com)

### Goal
The purpouse of this collaboration space is to trace and collect user feedback and make sure any issues or feature requests can be 
opned and detailed by everyone and that development can followed by our community: we want to directly work with you to help us design 
the features and tools that will make your life easier, while testing and scaling our infrastructure.
